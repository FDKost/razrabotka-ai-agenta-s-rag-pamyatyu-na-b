# Доработка AI‑агента с RAG‑памятью

## Requirements
- [high] Обновить создание агента: Заменить устаревший способ создания агента на вызов `create_agent` из LangChain, передав необходимые инструменты и параметры.
- [high] Установить системный промпт: При создании агента задать системный промпт, который явно указывает агенту использовать базу знаний и инструменты поиска/добавления.
- [high] Корректировать параметры чанкинга: В модуле разбиения документов задать `chunk_size=500` и `chunk_overlap=100` для `RecursiveCharacterTextSplitter`.
